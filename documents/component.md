# App 页面组件说明文档

## Tab 结构

| Tab | 页面 | 主要功能 |
|-----|------|----------|
| 首页 | Home | 轮播、导航、新闻 |
| 算力 | Mining | 机器购买、算力生产、签到打卡、算力转余额 |
| 任务 | Tasks | 任务列表、工分奖励 |
| 资产 | Assets | 余额查看、提现、账单记录 |
| 我的 | Profile | 用户信息、推广、账户设置 |

---

## 一、核心业务模块

### 1. 首页 (Home) - 信息展示入口

| 组件 | 说明 | 功能 |
|------|------|------|
| BannerCarousel | 轮播图组件 | 纯展示，无跳转链接 |
| NavGrid | 导航网格 | 快速入口：任务中心、邀请好友、签到打卡、我的团队 |
| NewsSection | 新闻中心 | 显示平台公告/新闻列表 |

---

### 2. 算力生产核心 (Mining) - 盈利核心

| 组件 | 说明 | 功能 |
|------|------|------|
| CPURentCard | CPU租用卡片 | 展示可购买机器（12代CPU），含价格、算力产出说明 |
| PaymentSuccessModal | 支付成功弹窗 | 支付完成确认 |
| CheckInButton | 签到打卡按钮 | 每日签到后启动已购CPU机器算力生产 |
| MiningDashboard | 算力生产面板 | 显示已购机器生产状态，24小时结算周期进度 |
| HashRateCalculator | 算力计算器 | 显示每小时产出（如1000级×0.03%=30算力/小时） |
| HashRateToBalanceCard | 算力转余额卡片 | 核心操作：将算力1:1转换为可提现余额 |
| MyMachinesList | 我的机器列表 | 显示已购买的所有机器及运行状态 |

**算力生产规则：**
1. 购买机器 → 支付成功 → 机器加入账户
2. 每日签到 → 启动算力生产
3. 每小时产生算力（级别×生产力%）
4. 24小时结算周期 → 算力累计
5. 算力转余额：算力1:1转为可提现余额（本金不可提，仅算力可转）

---

### 3. 任务中心 (Tasks) - 工分获取

| 组件 | 说明 | 功能 |
|------|------|------|
| TaskList | 任务列表 | 展示后台配置的任务（签到、邀请、首充） |
| TaskCard | 任务卡片 | 单条任务：描述、奖励工分数、完成状态 |
| WorkPointInfo | 工分说明 | 说明工分用途：提现必需材料 |
| DailyRefreshTip | 每日刷新提示 | 任务状态每日0点自动重置 |

**任务奖励表：**
| 任务类型 | 奖励 | 说明 |
|----------|------|------|
| 每日签到 | 工分 | 签到打卡同时启动算力生产 |
| 邀请好友注册 | 工分 | 后台配置奖励数量 |
| 每日首充 | 工分 | 后台配置奖励数量 |

**工分用途：** 每次提现必须扣除工分，作为提现"材料"

---

### 4. 交易/充值流程 (Product → Payment)

| 组件 | 说明 | 功能 |
|------|------|------|
| ProductDetailCard | 产品详情 | 12代CPU产品展示：价格、每小时收益、产品说明 |
| PaymentChannelList | 支付通道 | 支付宝、微信、快捷支付选项 |
| PaymentPasswordInput | 支付密码输入 | 6位数字密码确认 |
| ThirdPartyRedirect | 第三方支付跳转 | 跳转支付宝/微信/快捷支付页面 |
| PaymentResult | 支付结果页 | 成功/失败状态显示 |

**充值流程：**
选择产品 → 选择支付方式 → 输入支付密码 → 跳转第三方支付 → 完成支付 → 机器加入账户

---

## 二、用户功能模块

### 5. 资产页 (Assets) - 财务管理中心

**5.1 资产概览卡片**
| 组件 | 说明 | 功能 |
|------|------|------|
| WithdrawBalanceCard | 可提现余额 | 显示可提现总金额（算力转出+推荐奖励） |
| RechargeBalanceCard | 充值余额 | 显示充值总额（支付宝+微信+银行卡+快捷支付） |
| WorkPointCard | 工分卡片 | 显示当前工分数量（提现必需） |

**5.2 功能入口列表**
| 组件 | 说明 | 功能 |
|------|------|------|
| WithdrawEntry | 立即提现 | 跳转提现页面，扣除工分后提现 |
| WithdrawRecordEntry | 提现管理 | 查看所有提现记录及状态 |
| IncomeRecordEntry | 收益明细 | 查看每笔算力产出、推荐奖励记录 |
| OrderListEntry | 我的订单 | 查看所有产品购买订单（成功/失败） |
| MyProductsEntry | 我的产品 | 查看已购买的所有CPU机器及状态 |

---

### 6. 提现页 (Withdraw) - 资金提取

| 组件 | 说明 | 功能 |
|------|------|------|
| WithdrawCodeInput | 提款码输入 | 6位数字提款码验证 |
| SetWithdrawCode | 设置提款码 | 首次/忘记时设置新提款码 |
| BankCardSelector | 银行卡选择 | 选择提现到账银行卡 |
| WorkPointValidator | 工分验证 | 检查是否满足提现所需工分 |
| WithdrawAmountInput | 提现金额输入 | 输入提现金额 |
| WithdrawLimitTip | 提现限制提示 | 显示：最低提现额、每日次数限制、当前可提余额 |
| WithdrawSubmitButton | 提现提交 | 确认提现申请 |

**提现规则：**
1. 不能提取本金
2. 本金购买的机器产生的算力1:1转为余额，可提余额部分
3. 每次提现必须扣除工分
4. 后台配置限制：最低工分要求、最低提现额、每日提现次数

---

|------|------|------|

## 四、关键业务逻辑

### 资金流转
```
充值 → 购买机器 → 算力生产 → 算力转余额 → 提现到账
         ↑____________↓
           (每日签到启动)
```

### 工分流转
```
完成任务 → 获得工分 → 提现时扣除 → 再次完成任务补充
```

### 邀请层级
```
我 → 邀请好友A（一级）→ A邀请B（二级，我的二级）
```
